## ❓ Quick FAQ

**Is this really free?** The tool is free. AWS may charge for KMS key usage (first 20,000 requests/month free).

**What is a CSR?** Certificate Signing Request — a file sent to a Certificate Authority to request an SSL/TLS certificate.

**Why use AWS KMS for CSR?** Private keys never leave AWS KMS — more secure than software-based key generation.

**Do I need an AWS account?** Yes — you need an AWS account with a KMS key.

**What KMS keys are supported?** Symmetric (for encryption) and asymmetric (RSA, ECC) keys.

**Can I use an existing KMS key?** Yes — select any existing KMS key in your account.

**What are Subject Alternative Names (SANs)?** Domain names and IP addresses that the certificate should secure.

**Can I import the CSR to ACM?** Yes — direct integration with AWS Certificate Manager.

**Is this safe to use?** Yes — no malware, uses official AWS SDK, private keys stay in KMS.

**What if CSR generation fails?** Check IAM permissions and KMS key status.
